# oim

A set of common functions written in ES6 for personal usage.

## Installation

```bash
npm i @vikiboss/oim
```

## Usage

```javascript
const oim = require('@vikiboss/oim');

console.log(oim.shuffle(['o', 'i', 'm']));
```

> Full list of the functions see [list](./oim.d.ts)
